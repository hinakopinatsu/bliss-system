# BLISS（業務管理システム）

## 概要
AIESECにおける業務管理を効率化するためのアプリケーションです。

## 背景・課題
従来、渡航者情報や満足度調査はGoogleフォームで分散管理されており、データの一元管理ができていませんでした。  
また、クラウドサインとのAPI連携においてエラーが発生し、業務効率が低下していました。

## 解決したこと
・フォーム機能の内製化  
・データの一元管理  
・ユーザーごとの権限設計  
・API連携エラーの修正  

## 工夫した点
既存機能への影響を最小限に抑えるため、機能追加の形で設計しました。  
また、データの流れや構造を整理した上で実装を行いました。

## 使用技術
・Bubble  
・API連携  

## 学んだこと
複雑なシステムにおいては、構造を理解した上で影響範囲を考慮しながら改善することの重要性を学びました。
